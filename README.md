# ほいくシフト — 保育園シフト管理PWA

保育園の職員シフトを作成し、時間帯ごとの人員不足を自動で警告するアプリです。
職員ごとの担当クラス・勤務時間・勤務曜日をもとに、シフトを自動配置することもできます。
オフラインで動作し、スマホのホーム画面に追加して通常のアプリのように使えます。

## 入っているもの

- `index.html` … アプリ本体
- `manifest.webmanifest` … ホーム画面追加（インストール）用の設定
- `sw.js` … オフライン動作用のサービスワーカー
- `icon-192.png` / `icon-512.png` / `icon-maskable-512.png` … アプリアイコン

## できること

- 月カレンダー … 各日に配置人数を表示。日ごとの状態を色で表示（緑＝充足／黄＝ぎりぎり／赤＝人員不足）。各日の下の細いバーは7〜18時の時間帯ごとの充足状況。
- 日の詳細 … 時間帯別の充足グラフと、不足している時間帯（例「9時台 2/4名（あと2名）」）を自動で警告。
- 職員ごとの設定 … 設定画面で職員をタップすると、担当クラス・勤務できる時間帯（早番/中番/遅番、複数可）・勤務できる曜日を登録できます。
- 自動配置 … 上の職員設定と「時間帯ごとの必要人数」をもとに、不足を埋めるようシフトを自動生成します。
  - 日の詳細画面の「⚡ 自動配置で埋める」… その日の不足を埋めます。
  - カレンダー画面の「⚡ 今月の空きを自動配置」… 月内の各日（日曜は休園として除外）をまとめて埋めます。
  - 既存のシフトは消さず、足りない分だけ追加します。職員が足りず埋めきれない場合は、残る不足を表示します。
- シフト追加・編集 … 右下の「＋」から職員・日付・勤務パターンまたは任意の時刻・担当クラスを手動で設定。シフトをタップすると編集・削除。
- 設定 … 職員の追加・削除・編集、時間帯ごとの必要人数の調整。
- データはこの端末（ブラウザ）内に保存され、オフラインでも使えます。

## 自動配置の仕組み（かんたん）

「必要人数」と現在の配置を時間帯ごとに比べ、不足が大きい時間帯から順に、その時間帯をカバーでき、かつ勤務できる曜日・時間帯の条件に合う職員を割り当てていきます（1人1日1シフト）。各職員には登録した担当クラスが自動で入ります。あくまで下書きを素早く作るための機能なので、生成後に手動で微調整できます。

## 使い方（重要）

オフライン動作とホーム画面追加には、`https://` または `localhost` での配信が必要です
（`file://` で直接開くとアプリ画面とデータ保存は動きますが、インストールとオフライン機能は無効です）。

### いちばん簡単な確認方法（PCで）

このフォルダの中で以下を実行し、表示されたアドレスを開きます。

    python3 -m http.server 8000

→ ブラウザで `http://localhost:8000` を開く

### スマホで使う・ホーム画面に追加する

1. これらのファイルを HTTPS のWebサーバーに置く（GitHub Pages、Netlify、社内サーバーなど）。
2. スマホのブラウザでそのURLを開く。
3. iPhone（Safari）… 共有ボタン →「ホーム画面に追加」
   Android（Chrome）… メニュー →「アプリをインストール」、または設定画面の「ホーム画面に追加」ボタン。

これで通常のアプリのようにアイコンから起動でき、オフラインでも動作します。

## カスタマイズ

- 開園時間は `index.html` 内の `OPEN`（7時）・`CLOSE`（19時）で変更できます。
- 勤務パターン（早番・中番・遅番の時刻）は `PRESETS` で変更できます。
- 担当クラスの選択肢は `CLASSES` で変更できます。
- 初期表示の職員・サンプルシフトは `seed()` 関数にあります（実際の利用時は設定画面から自園の職員を登録してください）。
