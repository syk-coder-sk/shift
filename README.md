# ほいくシフト — 保育園シフト管理PWA

保育園の職員シフトを作成し、時間帯×担当クラスごとの人員不足を自動で警告するアプリです。
職員ごとの担当クラス・勤務時間・勤務曜日・希望休をもとに、シフトを自動配置できます。
オフラインで動作し、スマホのホーム画面に追加して通常のアプリのように使えます。

## 入っているもの

- `index.html` … アプリ本体
- `manifest.webmanifest` … ホーム画面追加（インストール）用の設定
- `sw.js` … オフライン動作用のサービスワーカー
- `icon-192.png` / `icon-512.png` / `icon-maskable-512.png` … アプリアイコン

## 画面（下のタブ）

- カレンダー … 月全体。各日の配置人数・状態（緑＝充足／黄＝ぎりぎり／赤＝人員不足）・希望休の人数を表示。「⚡ 今月の空きを自動配置」で月内をまとめて埋められます。
- 今日 … その日の詳細。時間帯×クラスの充足グラフと、不足を「9:00台 1歳児 1/2名（あと1）」のようにクラス別で警告。希望休の職員も表示。「⚡ 自動配置で埋める」「クリア」あり。
- 勤怠 … 月ごとの職員別の実働時間とシフト本数を一覧・バーで表示。実働時間は休憩（6時間超で45分、8時間超で60分）を自動で差し引きます。
- 設定 … 職員／勤務パターン／クラス別の必要人数／希望休の管理。

## 主な機能

### 職員ごとの設定（設定 → 職員をタップ）
- 担当クラス … その人のメインのクラス
- 勤務できる時間帯 … 勤務パターンから複数選択
- 勤務できる曜日 … 月〜日から選択

### 勤務パターンの編集（設定 → 勤務パターン）
- 早番・中番・遅番などの名称と時間を自由に追加・編集・削除できます。

### クラス別×時間帯の必要人数（設定 → クラス別の必要人数）
- クラスごとに、各時間帯（7〜18時台）に最低何名出勤させたいかを設定します。
  例：7時台は0歳児1名・1歳児2名…のように細かく指定できます。
- ここで設定した人数と実際の配置を比べ、不足を警告します。

### 希望休（設定 → 希望休）
- 職員・日付・理由を登録します。
- 希望休の職員は自動配置の対象から外れます（その日に割り当てられません）。
- 手動でシフトを入れようとした場合は確認メッセージが出ます。

### 自動配置
- 「クラス別の必要人数」と現在の配置を時間帯ごとに比べ、不足が大きいところから順に、
  条件（担当クラス・勤務できる曜日／時間帯・希望休でない）に合う職員を割り当てます（1人1日1シフト）。
- 埋めきれない場合は「まだ◯時間帯が不足」と表示します。生成後に手動で微調整できます。

### 作り直し（再配分）
- 自動配置は「空きを埋める」だけで既存シフトの入れ替えはしませんが、後から職員を追加したり必要人数を変えた場合に、
  きれいに反映させたいときは「作り直す」を使います。
  - カレンダー画面の「♻ 月を作り直す」… その月の自動配置分を削除し、現在の条件で配分し直します。
  - 日の詳細画面の「♻ 作り直す」… その日を配分し直します。
- 手動で追加・編集したシフトは作り直しても残ります（自動配置で作られた分だけが対象）。
  ※シフトを一度でも編集すると「手動」扱いになり、以降は作り直しの対象外（＝保持）になります。

### 時間帯の平等配分（自動配置の条件）
- 設定の「自動配置の条件 → 時間帯を平等に配分」をオンにすると、早番・遅番などが特定の人に偏らないよう、
  月内の担当回数をならして配分します。
- 人員の充足が最優先で、同じくらい不足を埋められる候補が複数いるときに、その時間帯の担当回数が少ない人を選びます。
  そのため、ある職員が1つの時間帯しか勤務できない設定なら、その人はその時間帯に集中します（設定上の制約が優先）。
- 各職員の時間帯ごとの担当回数は「勤怠」画面で確認できます（例：早番×11 中番×11）。

## 使い方（重要）

オフライン動作とホーム画面追加には、`https://` または `localhost` での配信が必要です
（`file://` で直接開くとアプリ画面とデータ保存は動きますが、インストールとオフライン機能は無効です）。

### PCで確認する

フォルダ内で次を実行し、表示されたアドレスを開きます。

    python3 -m http.server 8000

→ ブラウザで `http://localhost:8000`

### スマホで使う・ホーム画面に追加する

1. これらのファイルを HTTPS のWebサーバーに置く（GitHub Pages、Netlify、社内サーバーなど）。
2. スマホのブラウザでそのURLを開く。
3. iPhone（Safari）… 共有 →「ホーム画面に追加」／ Android（Chrome）… メニュー →「アプリをインストール」。

### 更新が反映されないとき（重要）
ブラウザがオフライン用に古い版を保持していることがあります。最新版に差し替えたのに変わらない場合は、
ページを再読み込み（必要なら「スーパーリロード」／キャッシュ消去）してください。
このアプリのサービスワーカーは、オンライン時は毎回最新のページを取りに行く設定にしてあるため、
通常は再読み込みで最新になります。

## カスタマイズ（コードを直接編集する場合）

- 開園時間は `index.html` 内の `OPEN`（7時）・`CLOSE`（19時）。
- クラスの選択肢は `CLASSES`。
- 初期データ（職員・勤務パターン・必要人数・サンプル希望休）は `seed()` 関数。
  実際の利用時は設定画面から自園の内容に置き換えてください。
